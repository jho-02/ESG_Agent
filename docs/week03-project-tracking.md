# Week 03 — Project 관리 및 이슈 체계

내용:
- GitHub Projects 칸반 컬럼: Backlog / To Do / In Progress / Review / Done
- 라벨 정책 예시(우선순위, 유형, 영역)
- 이슈 템플릿과 PR 템플릿
- 프론트엔드 작업 중심의 이슈 예시 10개

예시 이슈:
생성한 이슈 예시:
- [Bug] ESG 지표값이 잘못된 위치에 표시되는 문제
- [Bug] 화면 내 텍스트 크기가 일관되지 않은 문제
- [Feature] ESG 대시보드 메인 화면 구성
- [Feature] 기업별 상세 조회 UI 구현
- [Feature] ESG 점수 카드 컴포넌트 구현
- [Feature] 챗봇 기능 도입을 위한 채팅란 구현
- [Feature] 근거자료 및 전체 자료 다운로드 영역 구현
- [Feature] 사용자 친화형 UI 개선
- [Feature] UI 디자인 개선 및 전문가 검토 반영
- [Feature] ESG 평가 데이터 관리를 위한 DB 추가 구축
- [Chore] GitHub Actions CI 설정 정리
